Regression Model

KAN Model Architecture

Width: The width parameter defines the structure of the KAN in terms of input dimensions and the number of neurons in each layer. In the given example:

  • width=[3, 1]: This specifies that the model takes 3 input features and outputs a single value. This is aligned with the three features selected ('Days for shipping (real)', 'Days for shipment (scheduled)', 'Sales per customer') and the target being 'Benefit per order'.

Grid: The grid parameter controls the resolution of the tessellation grid used by KAN. Each input dimension is divided into a number of intervals equal to the grid size.

  • grid=5: This means that each feature is divided into 5 intervals. The KAN will use these intervals to partition the input space and fit local models (polynomials) within each partition. This helps in modeling non-linear relationships without needing a massive number of training data points.

k: The k parameter denotes the degree of the polynomial used in each partition.

  • k=3: This suggests that within each of the partitions created by the grid, a third-degree polynomial is used to model the relationship between the inputs and the output. Higher degrees allow the model to capture more complex patterns but can also lead to overfitting if not controlled properly.

def acc(formula, X, y):
    batch = X.shape[0]
    correct = 0
    for i in range(batch):
        correct += np.round(np.array(formula.subs('x_1', X[i,0]).subs('x_2', X[i,1])).astype(np.float64)) == y[i,0]
    return correct/batch
